# NOSTALGIC FIND 🎌

**Premium Vintage Thrift Store Online Platform**

A modern, fully-responsive web application for buying and selling curated vintage clothing. Perfect for online vintage thrift shops.

---

## ✨ Features

### For Customers:
- 🛍️ Browse vintage items by category
- 🔍 Filter and sort items
- 📱 Fully responsive (mobile, tablet, desktop)
- 💳 Clear ordering instructions
- 🖼️ Image galleries with zoom functionality
- 🎨 Beautiful, modern UI design

### For Admin:
- 📝 Add new items with images
- 📊 Inventory management dashboard
- 💰 Sales tracking and analytics
- 🏷️ Mark items as sold
- ✏️ Edit item details
- 🗑️ Delete items
- 📈 View recent sales
- 💾 Persistent storage (IndexedDB)

## 💾 Data Storage

- **Frontend:** IndexedDB (browser storage)
- **Fallback:** LocalStorage
- All data persists between sessions
- No backend server required!
